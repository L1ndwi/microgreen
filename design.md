# GrowEase Design Style Guide

## Design Philosophy

**Color Palette**
- Primary Green: #22c55e (fresh, natural growth)
- Secondary Green: #16a34a (deeper, mature growth)
- Accent Sage: #84cc16 (earthy, organic feel)
- Neutral White: #ffffff (clean, modern)
- Light Gray: #f8fafc (subtle backgrounds)
- Dark Gray: #1e293b (text, contrast)

**Typography**
- Display Font: Inter (bold, modern sans-serif)
- Body Font: Inter (clean, readable)
- Headings: 700 weight for impact
- Body text: 400 weight for readability

**Visual Language**
- Clean, eco-modern aesthetic
- Rounded corners and soft shadows
- Natural, organic shapes
- Subtle gradients and depth
- Emphasis on growth and freshness

## Visual Effects

**Used Libraries**
- Anime.js: Smooth animations and transitions
- Typed.js: Typewriter effect for hero headline
- Splitting.js: Text animation effects
- ECharts.js: Data visualization (if needed)
- p5.js: Creative coding for hero background
- Splide.js: Image carousels

**Effects & Animations**
- Typewriter animation for main headline
- Floating particles representing growth/seeds
- Smooth scroll reveal animations
- Hover effects with 3D transforms
- Pulse animation on CTA buttons
- Parallax scrolling for depth

**Styling Approach**
- Tailwind CSS for utility-first styling
- Custom CSS for advanced animations
- Mobile-first responsive design
- Dark mode support (future enhancement)

**Header Effect**
- Animated background with floating organic particles
- Gradient overlay with subtle movement
- Hero image with Ken Burns effect

**Interactive Elements**
- Smooth hover transitions (300ms ease)
- Form input focus states with green accent
- Button press animations with scale effect
- Card lift effects on hover

**Scroll Motion**
- Elements fade in when 50% visible
- Staggered animations for card grids
- Smooth easing curves (cubic-bezier)
- Maximum 24px vertical translation
